Important Concepts

Lesson 1: The Basics of the Web and HTML

WWWW - World Wide Web

The web uses HTML(Hyper Text Markup Language) and is made up of hyper links. It was invented in the 1990's and has over 30 billion pages.

Many types of files can be found on the web such as plain text, HTML, images, videos and music.

There are many components that make up the web. These components include the user, the users computer, the Internet, servers, etc...

HTML - Hypertext Markup Language

HTML is made up of text, markup, references to other documents, and links to other pages.

HTML markup it made of tags, an opening tag and a closing tag. Here are some tag examples: <p>, <b>, <em>, <b>, <div>, <span> etc...

There are different categories of tags block and inline tags are the two most used types. Block tags normally start and end with a new line. Examples of block tags: <p>, <div> <span>. Inline elements do not have line breaks. Some examples: <b>, <em>, <a>.

HTML's "tree-like" Structure

Like other programming languages HTML has a tree-like structure. Meaning the code branches off, that can be like hierarchical in nature.

Avoiding Repetition

Avoiding repetition is a good coding practice for many reasons. Just to name a few: If you have to change the code it is better and less errors to change it in one place instead of multiple places, easier to read code, smaller file size, less typing etc... This provides consistency to your code.

CSS - Cascading Style Sheet

HTML is used to provide structure to the web page. CSS is used to style the web page.

CSS allows you to use specific syntax and rules to make the web page look a certain way.

CSS can be inline(within the tag), internal(in the same file), and external(separate file).
